How to replace with your own Excel

  1. Place your .xlsx file in the same directory as this page or in the files/ subfolder.
  2. Change the href in the button above to your file path, e.g. href="yourfile.xlsx" or href="files/BossReport.xlsx".
  3. Keep the download attribute in the <a> tag so the browser triggers a direct download.

Tip: If you host on GitHub Pages / Netlify, these static files will be directly available for download.
